# 문제 해결 가이드

## 포트 충돌 오류 (EADDRINUSE)

### 증상
```
Error: listen EADDRINUSE: address already in use 0.0.0.0:3001
```

### 원인
포트 3001이 이미 다른 프로세스에 의해 사용 중일 때 발생합니다. 주로:
- 이전에 실행한 서버가 아직 종료되지 않음
- 여러 개의 서버 인스턴스가 동시에 실행됨
- 다른 애플리케이션이 포트 3001을 사용 중

### 해결 방법

#### 방법 1: 실행 중인 프로세스 종료
```bash
# 포트 3001을 사용하는 프로세스 찾기
lsof -ti:3001

# 프로세스 종료
kill -9 $(lsof -ti:3001)

# 또는 특정 PID 종료
kill -9 <PID>
```

#### 방법 2: 다른 포트 사용
`.env` 파일에서 포트를 변경:
```env
PORT=3002
```

#### 방법 3: 서버 재시작
```bash
# 모든 서버 프로세스 종료 후 재시작
pkill -f "tsx.*server"
npm run server:dev
```

### 예방 방법
- 서버를 종료할 때는 `Ctrl+C`로 정상 종료
- 백그라운드로 실행한 프로세스는 `ps` 명령어로 확인 후 종료
- 개발 시 하나의 터미널에서만 서버 실행
